# Introduction 
Enable or disable Overlay Filesystem or Write protect Boot partition via CLI

# Getting Started
1.	Needs root priviliges: use sudo
2.  Make executable: chmod +x overlayFS.sh
3.	run: sudo ./overlayFS.sh  -o y -b y
